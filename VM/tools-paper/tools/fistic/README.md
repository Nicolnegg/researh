# Fistic

**fistic** is a fault injection simulator.  The faults are simulated by code mutation.

The simulator was developped during the PhD of Thierno Barry, 2017.
The simulator was updated by Yanis Sellami during the development of PyAbduce in 2021-2023.

Targeted architecture: armv7

## Requirements

Python 3, Pip, qEMU for evaluation

## Installation

Run `./init.sh` and follow instructions.
